# HUB75
raspberryPiでHUB75規格のLEDマトリックスディスプレイを光らせる

# Dependency
https://github.com/WiringPi/WiringPi

# Usage
```sh
gcc -o main main.c gpio.c -lwiringPi
sudo ./main
```
